Proses Perhitungan KNN

Jalankan proses klasifikasi K-Nearest Neighbor untuk menghasilkan rekomendasi jurusan siswa.

Jalankan Algoritma KNN
Sistem akan menghitung jarak Euclidean Distance antara data siswa dan data alumni.
Tahapan Proses KNN
  • Mengambil data nilai siswa
  • Mengambil data alumni sebagai data latih
  • Menghitung Euclidean Distance
  • Mengambil tetangga terdekat
  • Menentukan hasil rekomendasi jurusan
Perhatian: Pastikan data siswa dan data alumni sudah diupload sebelum menjalankan proses KNN.
