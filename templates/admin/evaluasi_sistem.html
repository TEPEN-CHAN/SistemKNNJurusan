Keterangan Evaluasi
Confidence menunjukkan tingkat keyakinan sistem berdasarkan dominasi label jurusan dari tetangga terdekat. Semakin tinggi confidence, semakin kuat hasil rekomendasi yang diberikan.

Rata-rata jarak menunjukkan rata-rata kedekatan data siswa dengan data alumni terdekat. Semakin kecil jaraknya, semakin mirip data siswa dengan data latih alumni. Perhitungan jarak menggunakan variabel nilai mapel, minat bakat, dan lanjut PT.
Accuracy, Precision, Recall, dan F1-Score dihitung menggunakan data alumni sebagai data evaluasi. Sistem menguji setiap data alumni terhadap data alumni lainnya, lalu membandingkan hasil prediksi dengan jurusan asli pada kolom hasil_jurusan.
